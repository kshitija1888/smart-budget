# smart-budget
💸 Smart Budget — Personal Finance Management Web App

Smart Budget is a modern, user-friendly financial management web application designed to help users track expenses, manage budgets, and achieve financial goals efficiently.

🌟 Features
🔐 Authentication System
Secure login and signup interface for personalized user experience
📊 Interactive Dashboard
Visual overview of balance, expenses, savings, and goal progress
💳 Transaction Management
Add, view, search, and categorize expenses
🎯 Goal Tracking
Set financial goals and monitor progress with visual indicators
📈 Analytics & Insights
Charts and reports to analyze spending patterns
🤖 AI Financial Assistant
Smart suggestions and insights to improve saving habits
🏠 Household Budgeting
Manage shared expenses and family spending
🏆 Gamification & Achievements
Earn badges and track milestones
🌙 Dark Mode Support
Seamless light/dark theme switching
📤 Export Functionality
Download reports in PDF and CSV formats
